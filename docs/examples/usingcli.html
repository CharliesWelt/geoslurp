Using the command line tool geoslurp¶

A command line program is provided which can be used to register known datasets, and edit configuration settings. The full options are described in clitools/geoslurper.py, but the examples below can be a practical starting point for common tasks.

Getting info and help¶

To list available datasets:

geoslurper.py --list

To retrieve registration info on the current database holdings:

geoslurper.py --info
